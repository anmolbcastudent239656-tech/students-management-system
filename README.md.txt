# Student Management System

## Project Overview
This is a simple **console-based Python project** to manage student records.  
It allows users to **Add Students, View Students, and Search Students** using a simple menu-driven interface.  
Student data is stored in a **text file** for persistent storage.

---

## Features
- Add a new student (Name, Roll Number, Course)
- View all students
- Search for a student by Roll Number
- Simple and user-friendly menu interface

---

## Technology Used
- **Python 3.x**
- **File Handling (Text files)**
